1. Для закрепления темы компонентов:
	
	- сделайте с нуля ДЗ №3, используя vue-cli и сборку проекта
	(сначала делаем без иконок font-awesome, про них пункт №3)

2. Под прогресс бар сделайте отдельный компонент.
	Этот пункт ДЗ творческий, вам решать, какие он будет принимать параметры, 
	насколько много их будет и т.п.
	
	Данный компонент используем в App.vue для показа прогресса над формой.
	
3. Подключите svg-иконки https://www.npmjs.com/package/@fortawesome/vue-fontawesome
	
	Экосистема vue 3 пока что находится в состоянии становления, что тут без приключений не обойтись.
	По докам всё заработает, если обойдёте один косяк авторов.
	Когда дойдёте до раздела Using Vue 3.x, появится строка npm i --save @fortawesome/vue-fontawesome@3
	Она пока что не работает, нужно написать npm i --save @fortawesome/vue-fontawesome@3.0.0-1

	Дальше по докам должно всё получится.
	Ориентируйтесь на самый простой набор иконок - free-regular-svg-icons.
	
----------------------------------------------------------------------------------------------------------
ДЗ отправляется в виде архива, в который попадают все папки и файлы проекта кроме node_modules :)

Перед архивацией просьба правильно собрать проект:
	- поставить в vue config опцию publicPath = ''
	- запустить npm run build
